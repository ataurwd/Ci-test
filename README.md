# Task Management App

A React + Tailwind CSS task management UI application.

## Features

- **Kanban Board**: Three-column layout (To Do, In Progress, Done)
- **Drag & Drop**: Move tasks between columns
- **Add Tasks**: Create new tasks with title, description, and priority
- **Priority Levels**: High, Medium, Low with color coding
- **Delete Tasks**: Remove tasks from the board

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Tech Stack

- React 18
- Tailwind CSS
- Vite
